# Text2Table
Converts a text file into a python table.

## How to Use
1. Download Text2Table.py
2. Make new text file called "lines.txt" in the same foler
3. Put in whatever text you want in the text file, with a new line seperating them (It's easier to copy-paste)
4. Run the script. It will print out a python table with your desired text.

>[!CAUTION]
>Requires TQDM, Use `pip install tqdm`
